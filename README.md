ТЗ Подготовить часть интернет-магазина с использованием React 

1. Подготовить JSON набор данных (17 товаров ) - Url картинки - название - тип размера (вес/объем) - размер - штрихкод - производитель - Бренд - описание - цена - тип ухода

2. Подготовить страницу Каталога товаров (список товаров) 
2.1. Шапка и подвал должны быть сверстаны (но функциональный там только блок с корзиной) 
2.2. Сортировка 4 варианта - по цене и по названию, по убыванию и возрастанию
2.3. Фильтр в горизонтальном меню уход за телом/ уход за руками / и т.д. 
2.4.1. Один товар может подходить к нескольким типам (например, одновременно уход за руками и уход за лицом) 
2.5. В фильтре слева разместить - фильтр по цене - по производителю (поиск, чекбоксы, показать все - поле поиск здесь фильтрует самих производителей) - Уход за телом (дублирует горизонтальное меню) 
2.6. Нажав на название можно перейти на карточку товара

3. Страница карточки товара. - хлебные крошки - в корзину и +/- активно

4. Корзина 
4.1. Блок в шапке обновляется при действиях пользователя 
4.2. На странице корзины кнопки активны. +/-, удалить из корзины

5. В корзине после кнопки "оформить заказ" сразу отображаем "Спасибо за заказ" (модальное окно) и очищаем корзину.
